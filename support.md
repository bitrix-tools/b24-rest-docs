# Поддержка и комьюнити для разработчиков

Мы регулярно обновляем документацию по REST API Битрикс24 и добавляем примеры кода для типовых задач. Но помощь можно получить не только здесь.

Ниже — список источников, где вы можете задать вопрос, найти решение и поучаствовать в жизни комьюнити.

{% note info "Как задавать вопросы в чатах" %}

1. Сначала поищите ответ в документации или через поиск.
2. Попробуйте решить задачу самостоятельно. Даже частичный результат поможет точнее описать проблему.
3. Задавайте конкретные вопросы. Вместо «Не получается добавить кнопку, что делать?», попробуйте «Я зарегистрировал обработчик в `CRM_DEAL_DETAIL_TAB`, проверил методом `placement.list` — виджет есть. Но пользователи в карточке сделки его не видят».
4. Если метод работает не так, как ожидалось, прикрепите пример запроса и ответа API.

{% endnote %}

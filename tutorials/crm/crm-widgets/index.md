# Как встраивать виджеты в CRM

{% note tip "" %}

Выберите инструмент для разработки с AI-агентом:

- используйте [Битрикс24 Вайбкод](../../../ai-tools/vibecode.md), чтобы создать приложение для Битрикс24 по описанию задачи без знания языков программирования. Агент напишет код и разместит приложение на сервере без ручной настройки хостинга
- используйте [MCP-сервер](../../../ai-tools/mcp.md), чтобы разрабатывать интеграцию через REST API в своем проекте. Агент будет обращаться к официальной REST-документации

{% endnote %}

Виджеты добавляют интерфейс приложения в CRM. Пользователь работает с карточкой в Битрикс24, а интерфейс приложения появляется внутри карточки: в пользовательском поле лида или на отдельной вкладке.

Доступны два сценария. Первый показывает, как добавить в лид поле, которое открывает интерфейс приложения. Второй показывает видео-урок и материалы по добавлению вкладки в карточку CRM.

В обоих сценариях нужен обработчик — страница приложения с публичным URL. Битрикс24 открывает этот URL в карточке CRM и передает обработчику данные вызова. В каждом сценарии указано, какие данные получает обработчик.

> Быстрый переход: [все сценарии](#choose-tutorial)
>
> Пользовательская документация: [CRM в Битрикс24](https://helpdesk.bitrix24.ru/open/5435795/)

## Как начать работу

1. Определите, где должен появиться интерфейс приложения: в поле лида или на вкладке карточки CRM
2. Подготовьте обработчик — страницу приложения с URL, доступным из внешней сети
3. Найдите нужный сценарий в таблице [Как выбрать сценарий](#choose-tutorial)
4. Проверьте, какие права доступа и scope указаны в выбранном сценарии
5. Выполните методы в порядке, который описан в сценарии
6. Завершите установку приложения и откройте карточку CRM, чтобы проверить вызов обработчика

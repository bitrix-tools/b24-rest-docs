# Цветовые темы страницы

{% note tip "" %}

Если вы разрабатываете интеграции для Битрикс24 с помощью AI-инструментов (Codex, Claude Code, Cursor), подключите [MCP-сервер](../../../sdk/mcp.md), чтобы ассистент использовал официальную REST-документацию.

{% endnote %}

В разделе перечислены коды цветовых тем для поля `THEME_CODE`.

Задать значение `THEME_CODE` можно при создании и обновлении страницы через методы [landing.landing.add](./methods/landing-landing-add.md) и [landing.landing.update](./methods/landing-landing-update.md).

Получить текущее значение `THEME_CODE` можно через метод [landing.landing.getadditionalfields](./methods/landing-landing-get-additional-fields.md), если поле заполнено.

## Что важно знать

- `THEME_CODE` задает готовую цветовую палитру страницы
- настройки шрифтов хранятся отдельно, в полях `THEMEFONTS_*`
- если нужно задать собственный цвет, используйте `THEME_USE = Y` и `THEME_COLOR = #RRGGBB`
- при миграции старых typo-настроек модуль сопоставляет часть цветовых тем и наборы шрифтов по коду темы

# Интерактивные блоки: обзор

Интерактивные блоки в `landing` добавляют клиентское поведение без отдельной разработки скриптов. Для таких блоков вы подключаете нужное расширение в [манифесте блока](../manifest.md), настраиваете разметку и атрибуты, после чего платформа инициализирует поведение автоматически.

> Быстрый переход: [все сценарии раздела](#all-scenarios)

## Связанные сценарии

**Манифест блока.** Тип интерактивного сценария определяется набором подключенных расширений в `assets.ext`. Манифест можно получить через [landing.block.getmanifestfile](../methods/landing-block-get-manifest-file.md).

**Методы блока.** Для анализа готовых шаблонов используйте [landing.block.getrepository](../methods/landing-block-get-repository.md), затем откройте манифест выбранного блока через [landing.block.getmanifestfile](../methods/landing-block-get-manifest-file.md).

**Специальные блоки.** Если задача относится к меню, навигации, поиску, картам или CRM-формам, откройте раздел [Специальные блоки](../special/index.md).

## Выбор сценария

Откройте подраздел, если нужно:

- подключить слайдер на базе `js-carousel` и настроить поведение через `data-*` атрибуты — [Слайдеры](./sliders.md),
- подключить просмотр изображений в галерее через `js-gallery-cards` и `data-fancybox` — [Галереи](./gallery.md),
- добавить счетчик обратного отсчета с датой завершения в `data-end-date` — [Счетчики обратного отсчета](./timer.md).

## Что учитывать при настройке

- Расширение `landing_gallery_cards` подключается с зависимостями `landing_core` и `landing_fancybox`, а `landing_carousel` и `landing_countdown` подключаются с `landing_jquery`.
- `landing_gallery_cards` работает по CSS-селектору `.js-gallery-cards` и элементам с HTML-атрибутом `data-fancybox`, при этом в режиме редактирования галерея принудительно отключается.
- `landing_carousel` работает по CSS-селектору `.js-carousel`, а в режиме редактирования зацикливание принудительно отключается, чтобы не создавать клоны слайдов.
- `landing_countdown` инициализирует узлы по CSS-селектору `.js-countdown`, реагирует на изменение HTML-атрибута `data-end-date` и повторно инициализируется после изменений карточек.
- При совместном использовании слайдера и галереи сначала подключайте `landing_carousel`, затем `landing_gallery_cards`.

# Навигация и заголовок

Для блоков этого типа не нужно указывать специальный `subtype` в манифесте. Достаточно добавить в разметку блока один из маркеров:

- `#title#`
- `#breadcrumb#`

Система заменяет их при обработке содержимого блока.

## Маркер `#title#`

В публичной части маркер `#title#` заменяется на заголовок текущей страницы.

Пример блока с заголовком:

```html
<section class="landing-block g-pt-20 g-pb-20">
    <div class="landing-title-container container g-font-size-12">
        #title#
    </div>
</section>
```

В превью и редакторе может использоваться стандартное текстовое значение.

## Маркер `#breadcrumb#`

В публичной части маркер `#breadcrumb#` заменяется на навигационную цепочку текущей страницы.

В публичной части цепочка строится через `getNavChain()`. В превью и редакторе система может использовать шаблон `chain_template.php` активного шаблона сайта и подставлять шаблонную разметку.

Пример блока с навигационной цепочкой:

```html
<section class="landing-block g-pt-20 g-pb-20">
    <div class="landing-breadcrumb-container container g-font-size-12">
        #breadcrumb#
    </div>
</section>
```

## Что важно учитывать

- достаточно, чтобы нужный маркер был в HTML блока
- если в блоке нет этих маркеров, специальная подстановка не выполняется

## Примеры штатных блоков

- `store.breadcrumb`
- `store.breadcrumb_dark_bg_text_left`
- `store.store_v3_menu_2`

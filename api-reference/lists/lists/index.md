# Работа с универсальными списками: обзор методов

С помощью универсальных списков можно вести учет любых объектов: документов, сертификатов, рекламных материалов и другого.

Создавать списки может только администратор Битрикс24. Он же настраивает права: указывает, кто из сотрудников может редактировать список, а кто не имеет к нему доступа.

> Быстрый переход: [все методы](#all-methods) 
>
> Пользовательская документация: [Пример работы со списком](https://helpdesk.bitrix24.ru/open/1409963/)

## Типы инфоблоков для списков

В Битрикс24 списки используют три типа инфоблоков:

-  `lists` — стандартные универсальные списки,
-  `lists_socnet` — списки рабочих групп и проектов,
-  `bitrix_processes` — служебный тип для данных бизнес-процессов.

Чтобы узнать тип существующего списка, используйте метод [lists.get.iblock.type.id](./lists-get-iblock-type-id.md).

## Рабочие группы и проекты

Универсальные списки интегрированы [в рабочие группы и проекты](../../sonet-group/sonet-group-create.md). Создавать свои списки можно внутри любой группы или проекта с различными уровнями доступа для участников.

## Бизнес-процессы

Бизнес-процессы в универсальных списках создают индивидуальный сценарий обработки элементов. Например, элемент списка содержит дату окончания сертификата. При добавлении такого элемента запускается бизнес-процесс. За N дней до даты окончания он автоматически создаст задачу ответственному сотруднику.

Управление бизнес-процессами выполняется с помощью групп методов [bizproc.workflow.*](../../bizproc/index.md) и [bizproc.task.*](../../bizproc/bizproc-task/bizproc-task-list.md).

{% endnote %}

## Права доступа

Права на весь список настраиваются для пользователей, групп и отделов. Задайте их при создании или обновлении списка через параметр `RIGHTS`. В этом параметре укажите массив, где ключ — это код сущности, например, `U{ID}` для пользователя, а значение — буквенный код права, например, `W` для записи.

Права могут конфликтовать, например, у пользователя есть роль с одними правами, а отдел, в котором он состоит, — с другими. В этом случае применяются максимальные права.

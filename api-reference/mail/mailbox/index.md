# Почтовые ящики: обзор методов

Почтовые ящики в Битрикс24 хранят параметры подключения почты пользователя и набор доступных адресов отправителя.

С помощью методов раздела можно:

- получить список подключенных ящиков
- запросить данные ящика по его идентификатору
- узнать доступные адреса отправителя
- просмотреть список полей ящика и описание конкретного поля

> Быстрый переход: [все методы](#all-methods)
>
> Пользовательская документация: [Способы подключения почтовых ящиков в Битрикс24](https://helpdesk.bitrix24.ru/open/19083990/)

Методы раздела относятся к REST 3.0. Особенности вызова и формат ответа новой версии API описаны в [обзоре REST 3.0](../../rest-v3.md).

{% endnote %}

## Когда использовать каждый метод

Используйте [mail.mailbox.list](./mail-mailbox-list.md), когда нужно получить список доступных ящиков и выбрать `id` для дальнейших запросов.

Используйте [mail.mailbox.get](./mail-mailbox-get.md), когда нужно получить параметры конкретного ящика по его `id`.

Используйте [mail.mailbox.senders](./mail-mailbox-senders.md), когда нужно получить список адресов отправителя, доступных текущему пользователю.

Используйте [mail.mailbox.field.list](./mail-mailbox-field-list.md) и [mail.mailbox.field.get](./mail-mailbox-field-get.md), когда нужно:
- узнать доступные поля почтового ящика
- получить типы и метаданные конкретного поля

## Порядок работы с почтовыми ящиками

1. Получите список ящиков через [mail.mailbox.list](./mail-mailbox-list.md)
2. Выберите нужный `id` и получите детали через [mail.mailbox.get](./mail-mailbox-get.md)
3. Получите адреса отправителя через [mail.mailbox.senders](./mail-mailbox-senders.md) перед отправкой письма
4. Уточните структуру полей через `*.field.list` и `*.field.get`

## Ограничения раздела

- Состав доступных ящиков и отправителей зависит от прав текущего пользователя
- Методы раздела не отправляют письма, а подготавливают данные для работы с письмами

# Получатели в почте: обзор методов

{% note tip "" %}

Выберите инструмент для разработки с AI-агентом:

- используйте [Битрикс24 Вайбкод](../../../ai-tools/vibecode.md), чтобы создать приложение для Битрикс24 по описанию задачи без знания языков программирования. Агент напишет код и разместит приложение на сервере без ручной настройки хостинга
- используйте [MCP-сервер](../../../ai-tools/mcp.md), чтобы разрабатывать интеграцию через REST API в своем проекте. Агент будет обращаться к официальной REST-документации

{% endnote %}

Получатели в почте Битрикс24 — это контакты и сотрудники, которых можно выбрать при адресации письма.

Методы раздела позволяют:
- искать контакты из адресной книги по строке поиска
- искать сотрудников по имени и e-mail
- получать список полей получателя и описание конкретного поля

> Быстрый переход: [все методы](#all-methods)
>
> Пользовательская документация: [Как работать с почтой в Битрикс24](https://helpdesk.bitrix24.ru/open/18258092/)

{% note info "" %}

Методы раздела относятся к REST 3.0. Особенности вызова и формат ответа новой версии API описаны в [обзоре REST 3.0](../../rest-v3.md).

{% endnote %}

## Когда использовать каждый метод

Используйте [mail.recipient.listcontacts](./mail-recipient-listcontacts.md), когда нужно подобрать внешних получателей из адресной книги для письма.

Используйте [mail.recipient.listemployees](./mail-recipient-listemployees.md), когда нужно найти сотрудника Битрикс24 по имени или e-mail.

Используйте [mail.recipient.field.list](./mail-recipient-field-list.md) и [mail.recipient.field.get](./mail-recipient-field-get.md), когда нужно:
- узнать доступные поля получателя
- получить типы и метаданные конкретного поля

## Порядок работы с получателями

1. Подберите адресатов письма: внешних получателей через [mail.recipient.listcontacts](./mail-recipient-listcontacts.md), сотрудников через [mail.recipient.listemployees](./mail-recipient-listemployees.md)
2. Проверьте структуру данных получателя: список полей через [mail.recipient.field.list](./mail-recipient-field-list.md) и описание поля через [mail.recipient.field.get](./mail-recipient-field-get.md)

## Ограничения раздела

- Результаты поиска зависят от поисковой строки и доступности данных текущему пользователю
- Методы раздела подбирают адресатов и не выполняют отправку писем

## Связь с другими объектами

**Письмо.** Методы раздела помогают выбрать адресатов для отправки и ответа на письма в методах [mail.message.*](../message/index.md).

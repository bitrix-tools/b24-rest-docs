Без параметров.

## Ответ в случае успеха

```json
{    
    "result": {
        "rest": 2,
        "web": 1,
        "mobile": 1,
    }
}
```

**Описание ключей**:

- `rest` – ревизия api для rest клиентов
- `web` – ревизия api для веб/десктоп клиента
- `mobile` – ревизия api для мобильного клиента

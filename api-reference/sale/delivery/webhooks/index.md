Битрикс24 оповещает службу доставки при расчете стоимости, создании заказа на доставку и отмене заказа. Для каждого действия Битрикс24 отправляет HTTP-запрос в формате JSON на URL, который указан в настройках обработчика службы доставки.

URL обработчиков задаются в параметрах метода [sale.delivery.handler.add](../handler/sale-delivery-handler-add.md). По этим URL внешняя служба доставки получает данные заказа и возвращает результат обработки в JSON.

> Быстрый переход: [все события](#all-events)
>
> Пользовательская документация: [Службы доставки](https://helpdesk.bitrix24.ru/open/17225250/)

## Как получать события

1. Создайте обработчик службы доставки методом [sale.delivery.handler.add](../handler/sale-delivery-handler-add.md)
2. В параметре `SETTINGS` укажите URL для расчета, создания и отмены заказа на доставку
3. Реализуйте обработку входящих HTTP-запросов в формате JSON
4. Верните JSON-ответ в формате, который описан на странице нужного события

## URL обработчиков

#|
|| **Параметр** | **Событие** | **Когда Битрикс24 отправляет запрос** ||
|| `CALCULATE_URL` | [Расчет стоимости доставки](./calculate.md) | Когда менеджер рассчитывает предварительную стоимость доставки ||
|| `CREATE_DELIVERY_REQUEST_URL` | [Создание заказа на доставку](./create-delivery-request.md) | Когда менеджер оформляет заказ на доставку ||
|| `CANCEL_DELIVERY_REQUEST_URL` | [Отмена заказа на доставку](./cancel-delivery-request.md) | Когда менеджер отменяет ранее оформленный заказ на доставку ||
|#

`CALCULATE_URL` обязателен при создании обработчика службы доставки. URL для создания и отмены заказа указывайте, если служба доставки поддерживает оформление и отмену заказов из Битрикс24.

## Формат обмена

Битрикс24 отправляет на URL обработчика HTTP-запрос с JSON-телом. Структура запроса зависит от события и описана на странице конкретного события.

Внешний обработчик должен вернуть JSON-ответ:

- успешный ответ подтверждает расчет, создание или отмену заказа
- ответ с ошибкой передает причину, которую можно показать менеджеру

## Доступность сервера обработчика

URL обработчиков должны быть доступны для входящих запросов от Битрикс24. Если сервер службы доставки недоступен или возвращает ответ не в ожидаемом JSON-формате, действие с доставкой завершится ошибкой.

# Удалить оплату sale.payment.delete

> Scope: [`sale`](../../scopes/permissions.md)
>
> Кто может выполнять метод: администратор

Метод удаляет оплату.

## Параметры метода

#|
|| **Название**
`тип` | **Описание** ||
|| **id**^*^
[`sale_order_payment.id`](../data-types.md) | Идентификатор оплаты ||
|#

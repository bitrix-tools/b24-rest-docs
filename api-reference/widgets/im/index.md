В рамках чата есть несколько точек встраивания:

- [IM_TEXTAREA](./textarea.md) – приложение для панели над полем ввода (данный формат был и раньше – это генерация контента в момент написания сообщения);
- [IM_SIDEBAR](./sidebar.md) – приложение для сайдбара (можно создавать приложения, добавляющие дополнительные сценарии для чата – например, отдельный диск для чата или базу знаний);
- [IM_CONTEXT_MENU](./context-menu.md) – приложение для открытия контекстного меню сообщения внутри чата, встраивание в пункт «Создать контент на основании» (аналогом является «Создать задачу» или «Создать встречу» на основании сообщения);

До версии `im 25.1600.0`: 
- [IM_SMILES_SELECTOR](./smile-selector.md) – приложение для расширения возможностей смайлов и giphy (тут могут быть свои источники картинок или смайлов).

{% note tip "" %}

В Битрикс24 можно подключить почтовые ящики от сервисов Яндекс, Mail.ru, Gmail и других. Это упрощает работу с письмами.

> Быстрый переход: [все методы](#all-methods)

Методы раздела управляют почтовыми сервисами. Почтовыми ящиками, письмами и получателями управляют методы [Почта REST 3.0](../mail/index.md).

## Как начать работу

1. Создайте почтовый сервис методом [mailservice.add](./mailservice-add.md)
2. Измените параметры сервиса методом [mailservice.update](./mailservice-update.md)
3. Получите сервис по идентификатору методом [mailservice.get](./mailservice-get.md)
4. Проверьте список активных сервисов методом [mailservice.list](./mailservice-list.md)
5. Удалите сервис методом [mailservice.delete](./mailservice-delete.md), если он больше не нужен
6. Чтобы проверить доступные поля, используйте метод [mailservice.fields](./mailservice-fields.md)

## Идентификаторы почтового сервиса

- `ID` — идентификатор почтового сервиса. Его возвращает метод [mailservice.add](./mailservice-add.md)
- Получить параметры сервиса по `ID` можно методом [mailservice.get](./mailservice-get.md)
- Найти активные почтовые сервисы и их `ID` можно методом [mailservice.list](./mailservice-list.md)

# Файлы: обзор методов

Раздел содержит методы для загрузки файлов в чат и получения ссылки на скачивание файла от имени пользователя или приложения.

> Быстрый переход: [все методы](#all-methods)

## Как работать с файлами

- используйте [im.v2.File.upload](./file-upload.md), чтобы загрузить файл в чат
- используйте [im.v2.File.download](./file-download.md), чтобы получить ссылку на скачивание уже загруженного файла

## Обзор методов {#all-methods}

> Scope: [`im`](../../../../scopes/permissions.md)
>
> Кто может выполнять методы: пользователь или приложение с доступом к мессенджеру

#|
|| **Метод** | **Описание** ||
|| [im.v2.File.upload](./file-upload.md) | Загружает файл в чат ||
|| [im.v2.File.download](./file-download.md) | Возвращает ссылку для скачивания файла ||
|#

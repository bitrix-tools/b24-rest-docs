# Блок со ссылками LINK

{% note tip "" %}

Выберите инструмент для разработки с AI-агентом:

- используйте [Битрикс24 Вайбкод](../../../../../../../ai-tools/vibecode.md), чтобы создать приложение для Битрикс24 по описанию задачи без знания языков программирования. Агент напишет код и разместит приложение на сервере без ручной настройки хостинга
- используйте [MCP-сервер](../../../../../../../ai-tools/mcp.md), чтобы разрабатывать интеграцию через REST API в своем проекте. Агент будет обращаться к официальной REST-документации

{% endnote %}

Блок `LINK` выводит ссылку с подписью, описанием и опциональным изображением-превью.

![Блок со ссылками](./_images/link.png){width=420}

## Когда использовать LINK

`LINK` подходит для ручного формирования ссылочного блока во вложении.

Если нужен именно формат расширенного предпросмотра ссылки, используйте `RICH_LINK`.

## Параметры блока

#|
|| **Название**
`тип` | **Описание** ||
|| **LINK***
[`string`](../../../../../../data-types.md) | URL ссылки. Допускаются абсолютные URL (`http://`, `https://`) и относительные пути от корня Битрикс ||
|| **NAME**
[`string`](../../../../../../data-types.md) | Текст ссылки. Если не задан, отображается `LINK` ||
|| **DESC**
[`string`](../../../../../../data-types.md) | Описание под заголовком ссылки ||
|| **HTML**
[`string`](../../../../../../data-types.md) | HTML-описание. Если задано, используется вместо `DESC` ||
|| **PREVIEW**
[`string`](../../../../../../data-types.md) | URL изображения-превью ||
|| **WIDTH**
[`integer`](../../../../../../data-types.md) | Ширина превью в пикселях ||
|| **HEIGHT**
[`integer`](../../../../../../data-types.md) | Высота превью в пикселях ||
|| **USER_ID**
[`integer`](../../../../../../data-types.md) | Ссылка на пользователя Битрикс (внутренняя навигация) ||
|| **CHAT_ID**
[`integer`](../../../../../../data-types.md) | Ссылка на чат Битрикс (внутренняя навигация) ||
|| **NETWORK_ID**
[`string`](../../../../../../data-types.md) | Ссылка на сетевого пользователя Bitrix24 Network ||
|#

## Пример

{% include [Сноска о примерах](../../../../../../../_includes/examples.md) %}

{% list tabs %}

- JS

    ```js
    {
        LINK: {
            PREVIEW: 'https://dev.1c-bitrix.ru/bitrix/templates/1c-bitrix-new/images/logo.png',
            WIDTH: 1000,
            HEIGHT: 638,
            NAME: 'Тикет #12345: новое API для модуля "Веб-мессенджер"',
            DESC: 'Необходимо реализовать к релизу!',
            LINK: 'https://api.bitrix24.com/'
        }
    }
    ```

- PHP

    ```php
    [
        'LINK' => [
            'PREVIEW' => 'https://dev.1c-bitrix.ru/bitrix/templates/1c-bitrix-new/images/logo.png',
            'WIDTH' => 1000,
            'HEIGHT' => 638,
            'NAME' => 'Тикет #12345: новое API для модуля "Веб-мессенджер"',
            'DESC' => 'Необходимо реализовать к релизу!',
            'LINK' => 'https://api.bitrix24.com/'
        ]
    ]
    ```

# Файлы: обзор методов

{% note tip "" %}

Выберите инструмент для разработки с AI-агентом:

- используйте [Битрикс24 Вайбкод](../../../../../ai-tools/vibecode.md), чтобы создать приложение для Битрикс24 по описанию задачи без знания языков программирования. Агент напишет код и разместит приложение на сервере без ручной настройки хостинга
- используйте [MCP-сервер](../../../../../ai-tools/mcp.md), чтобы разрабатывать интеграцию через REST API в своем проекте. Агент будет обращаться к официальной REST-документации

{% endnote %}

Методы позволяют загружать файлы в чат от имени бота и получать ссылку на скачивание уже загруженного файла.

> Быстрый переход: [все методы](#all-methods)

## Как работать с файлами

- используйте [imbot.v2.File.upload](./file-upload.md), чтобы загрузить файл в чат и при необходимости отправить его вместе с сообщением
- используйте [imbot.v2.File.download](./file-download.md), чтобы получить ссылку для скачивания файла

## Обзор методов {#all-methods}

>
> Кто может выполнять методы: владелец зарегистрированного бота

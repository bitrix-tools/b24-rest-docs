Раздел содержит методы для регистрации слэш-команд чат-бота, их изменения, удаления и отправки ответа на вызов команды.

> Быстрый переход: [все методы](#all-methods)

## Как работают слэш-команды

Слэш-команды позволяют вызывать функции бота двумя способами: 

- ввести `/команда` в поле ввода чата,
- вызвать команду через [кнопку клавиатуры](../messages/message-keyboards.md). 

Во втором случае команда используется как скрытое действие кнопки и не требует ручного ввода текста. После регистрации команды бот получает событие вызова и может ответить через [imbot.v2.Command.answer](./command-answer.md).

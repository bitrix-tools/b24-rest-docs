# Команды: обзор методов

Раздел содержит методы для регистрации слэш-команд чат-бота, их изменения, удаления и отправки ответа на вызов команды.

> Быстрый переход: [все методы](#all-methods)

## Как работают слэш-команды

Слэш-команды позволяют пользователю вызывать функции бота, вводя `/команда` в поле ввода чата. После регистрации команды бот получает событие вызова и может ответить через [imbot.v2.Command.answer](./command-answer.md).

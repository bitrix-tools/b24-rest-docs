# Бот-платформа: обзор раздела

Раздел объединяет документацию по API чат-ботов в Битрикс24. Здесь собраны как актуальные методы новой бот-платформы `imbot.v2`, так и устаревшие методы предыдущего поколения `imbot.*`.

{% note tip "" %}

Если вы разрабатываете интеграции для Битрикс24 с помощью AI-инструментов (Codex, Claude Code, Cursor), подключите [MCP-сервер](../../sdk/mcp.md), чтобы ассистент использовал официальную REST-документацию.

{% endnote %}

## Что умеют чат-боты в Битрикс24

Чат-боты могут:

- отправлять и изменять сообщения в чатах
- получать события и реагировать на действия пользователей
- работать со слэш-командами
- загружать и скачивать файлы
- управлять групповыми чатами и участниками
- использовать клавиатуры, вложения и другие элементы интерфейса

## Актуальная версия API {#v2}

[Чат-боты 2.0](./chat-bots-v2/index.md) — основной раздел для новой разработки. Он включает:

- бот-платформу `imbot.v2` для регистрации и управления ботами
- агентский режим `im.v2`, позволяет обрабатывать пользовательские события мессенджера
- новые методы для работы с чатами, сообщениями, файлами, командами и интерфейсом

Если вы начинаете создание нового Чат-бота, ориентируйтесь именно на этот раздел.

## Устаревшие методы {#outdated}

[Устаревшее API](./outdated/index.md) содержит документацию по предыдущей версии методов `imbot.*`, `imbot.chat.*`, `imbot.message.*`, `imbot.command.*` и связанным событиям.

Эти методы оставлены для поддержки существующих интеграций. Для новых решений строго рекомендовано использовать [Чат-боты 2.0](./chat-bots-v2/index.md).

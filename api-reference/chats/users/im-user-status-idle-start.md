Метод `im.user.status.idle.start` устанавливает автоматический статус «Отошел».

#|
|| **Параметр** | **Пример** | **Описание** | **Ревизия** ||
|| **AGO**
[`unknown`](../../data-types.md) | `10` | Сколько минут назад отошел | `18` ||
|#

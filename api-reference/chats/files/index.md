# Файлы в чатах: обзор методов

Сейчас базовый сценарий работы с файлами в чатах стал проще: для загрузки и скачивания файла не нужно отдельно получать папку чата, загружать файл через методы Диска и затем привязывать его к сообщению.

Для новых интеграций используйте методы `im.v2`:

- [im.v2.File.upload](../../chat-bots/chat-bots-v2/im.v2/files/file-upload.md) — загружает файл в чат
- [im.v2.File.download](../../chat-bots/chat-bots-v2/im.v2/files/file-download.md) — возвращает ссылку на скачивание файла

> Быстрый переход: [новые методы](#all-methods) | [устаревшие методы](#legacy-methods)

## Как работать с файлами сейчас

Типовой сценарий:

1. Загрузите файл в чат через [im.v2.File.upload](../../chat-bots/chat-bots-v2/im.v2/files/file-upload.md).
2. Используйте полученный результат в логике приложения или интерфейса чата.
3. Когда нужно отдать файл пользователю повторно, получите ссылку через [im.v2.File.download](../../chat-bots/chat-bots-v2/im.v2/files/file-download.md).

{% note info "" %}

Методы `im.v2.File.*` относятся к новому поколению API мессенджера. Их рекомендуется использовать в новых сценариях вместо цепочки вызовов `im.disk.*` и методов Диска.

{% endnote %}

## Что изменилось по сравнению со старым сценарием

Раньше для загрузки файла часто требовалась цепочка из нескольких действий:

- получить папку хранения файлов чата
- загрузить файл через методы Диска
- привязать загруженный файл к чату

Теперь для большинства сценариев достаточно прямых методов:

- [im.v2.File.upload](../../chat-bots/chat-bots-v2/im.v2/files/file-upload.md)
- [im.v2.File.download](../../chat-bots/chat-bots-v2/im.v2/files/file-download.md)

#|
|| **Параметр** | **Описание** ||
|| **start** | Порядковый номер элемента списка, начиная с которого необходимо возвращать следующие элементы при вызове текущего метода. Подробности в статье [{#T}](../../how-to-call-rest-api/list-methods-pecularities.md) ||
|#

> 200 OK

	[
		 "id": "1",
		 "name": "Администратор",
		 "code": "ADMIN",
	],
	[
		"id": "2",
		"name": "Менеджер",
		"code": "MANAGER",

}

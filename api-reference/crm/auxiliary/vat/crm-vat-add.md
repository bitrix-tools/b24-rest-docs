## Описание

```http
crm.vat.add(fields)
```

Метод создает новую ставку НДС.

## Параметры

#|
|| **Параметр** | **Описание** ||
|| **fields** | Набор полей - массив вида `array("поле"=>"значение"[, ...])`, содержащий значения ставки НДС. ||
|#

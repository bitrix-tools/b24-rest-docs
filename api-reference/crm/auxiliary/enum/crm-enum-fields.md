## Описание

```js
crm.enum.fields()
```

Метод возвращает описание полей перечисления.

## Параметры

Без параметров.

## Примеры

```javascript
BX24.callMethod(
    "crm.enum.fields",
    {},
    function(result)
    {
        if(result.error())
            console.error(result.error());
        else
            console.dir(result.data());
    }
);
```

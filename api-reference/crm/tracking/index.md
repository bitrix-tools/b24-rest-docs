# Сквозная аналитика в CRM: обзор методов

Сквозная аналитика в CRM позволяет связывать лиды, сделки, контакты, компании и коммерческие предложения с источниками и маршрутами привлечения.

Методы раздела помогают создавать и использовать трейсы сквозной аналитики в REST API CRM.

## Обзор методов {#all-methods}

> Scope: [`crm`](../../scopes/permissions.md)
>
> Кто может выполнять метод: любой пользователь

#|
|| **Метод** | **Описание** ||
|| [crm.tracking.trace.add](./crm-tracking-trace-add.md) | Создает трейс сквозной аналитики и возвращает его идентификатор ||
|| [crm.tracking.trace.delete](./crm-tracking-trace-delete.md) | Удаляет трейс сквозной аналитики ||
|#

Методы раздела работают с доставками, привязанными к универсальным объектам CRM. Например, можно получить список доставок для сделки или узнать детали конкретной отгрузки по лидy.

Это нужно, чтобы интегрировать логистические сервисы с данными CRM. Битрикс24 хранит информацию о доставке отдельно от основных полей объекта, но связывает их через идентификаторы.

> Быстрый переход: [все методы](#all-methods)

## Связь доставок с объектами CRM

**Объект CRM.** Связь осуществляется через пару параметров `entityTypeId` и `entityId`. Первый указывает на [тип объекта CRM](../../data-types.md#object_type), второй — на конкретный элемент этого типа.

**Служба доставки.** Каждая запись о доставке содержит ссылку на настроенную службу доставки. Идентификатор службы доставки можно получить методом [sale.delivery.getlist](../../../sale/delivery/delivery/sale-delivery-get-list.md). В текущем разделе `deliveryId` возвращают методы [crm.item.delivery.list](./crm-item-delivery-list.md) и [crm.item.delivery.get](./crm-item-delivery-get.md).

## Как работать с доставками в CRM

1. Определите тип объекта CRM `entityTypeId` и его идентификатор `entityId`.
2. Получите список доставок, которые связаны с объектом, методом [crm.item.delivery.list](./crm-item-delivery-list.md).
3. Выберите нужную доставку из списка и получите информацию по ее `id` методом [crm.item.delivery.get](./crm-item-delivery-get.md).

# Прикрепленный файл: обзор методов

Прикрепленный файл связывает документ из Диска с другими объектами Битрикс24. Эта связь обеспечивает доступ к файлу из разных модулей системы.

> Быстрый переход: [все методы](#all-methods) 

## Особенности работы с прикрепленными файлами

Метод `disk.attachedObject.get` работает с записью о прикреплении файла к конкретному объекту, а не с самим файлом. Представьте, что у вас есть файл `report.docx` на Диске. Когда вы прикрепляете его к задаче и к комментарию в ленте, создаются две разные записи о прикреплении с разными идентификаторами, но обе ссылаются на один и тот же файл.

Для получения идентификатора связи необходимо использовать методы, которые возвращают прикрепленные файлы в рамках конкретных объектов. Например, при работе с задачами идентификатор связи можно получить через метод [tasks.task.get](../../tasks/tasks-task-get.md).

## Связь с другими объектами

**CRM.** Файлы можно прикрепить к делам и коммерческим предложениям. За работу с делами отвечает группа методов [crm.activity.\*](../../crm/timeline/activities/index.md), за работу с предложениями — [crm.quote.\*](../../crm/quote/index.md).

**Бизнес-процессы.** Можно запустить для документов общего диска. Управление бизнес-процессами выполняется методами [bizproc.workflow.\*](../../bizproc/index.md).

**Задачи.** Файлы прикрепляются к описанию задачи. Все участники задачи могут просматривать, редактировать и скачивать прикрепленные файлы. Работать с задачами необходимо через методы группы [tasks.task.\*](../../tasks/index.md).

**Календарь.** Файлы добавляются к событию и становятся доступными всем участникам. Получить информацию о событии можно с помощью методов [calendar.event.\*](../../calendar/index.md).

**Лента новостей.** Файлы прикрепляются к сообщениям и комментариям. Метод [log.blogpost.get](../../log/log-blogpost-get.md) возвращает список сообщений, а метод [log.blogcomment.user.get](../../log/blogcomment/log-blogcomment-user-get.md) — список комментариев.

**Почта.** Вложения из писем сохраняются на Диске. С письмами можно работать только через интерфейс Битрикс24. Если письмо сохранено в CRM, то с вложениями можно взаимодействовать методами [crm.activity.\*](../../crm/timeline/activities/index.md) через поле `FILES`.

**Универсальные списки.** Элементы списка связаны с Диском через [поле](../lists/fields/index) типа Файл (Диск). Получить параметры элемента можно с помощью метода [lists.element.get](../../lists/elements/lists-element-get.md).

**Чаты.** Пользователи могут обмениваться документами, фото, видео и аудио. Файлы можно просматривать и скачивать, а документы — редактировать без скачивания. Прикрепленные файлы доступны всем участникам чата. Метод [im.dialog.messages.get](../../chats/messages/im-dialog-messages-get.md) возвращает список последних сообщений.

> Scope: [`disk`](../../scopes/permissions.md)
>
> Кто может выполнять метод: пользователь с правом «Чтение» для нужного файла

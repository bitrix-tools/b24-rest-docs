Версия файла — это сохраненный снимок файла на определенный момент времени. При изменении файла система создает новую версию. Это позволяет отслеживать историю изменений и восстанавливать файлы до нужного состояния.

> Быстрый переход: [все методы](#all-methods) 
>
> Пользовательская документация: [Сколько хранятся версии документа на Диске](https://helpdesk.bitrix24.ru/open/18869612/) 

## Порядок работы с версиями файлов

Метод [disk.version.get](./disk-version-get.md) возвращает информацию о версии файла, включая ее размер, время создания, идентификатор пользователя, создавшего эту версию, а также временную ссылку для скачивания версии.

Чтобы получить данные о версии:

1. Запросите список версий файла методом [disk.file.getVersions](../file/disk-file-get-versions.md). В ответе вы получите массив с `ID` всех доступных версий.

2. Используйте нужный `ID` как параметр в методе [disk.version.get](./disk-version-get.md).

Метод `disk.file.markdeleted` перемещает файл в корзину.

## Параметры

#|
||  **Параметр** / **Тип**| **Описание** ||
|| **id**
[`unknown`](../../data-types.md) | Идентификатор файла. ||
|#

## Пример

{% list tabs %}

- JS

    ```js
    BX24.callMethod(
        "disk.file.markdeleted",
        {
            id: 10
        },
        function (result)
        {
            if (result.error())
                console.error(result.error());
            else
                console.dir(result.data());

    );
    ```

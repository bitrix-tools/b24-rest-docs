|| **id^*^**

|#

## Возвращаемое значение

Возвращает json массив данных о задаче (аналогично методу [`tasks.task.get`](./tasks-task-get.md)).

## Пример

```js
BX24.callMethod('tasks.task.unmute', {id: 1223})
```

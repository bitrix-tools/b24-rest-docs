# Действия пользователя по задаче: обзор методов

{% note tip "" %}

Выберите инструмент для разработки с AI-агентом:

- используйте [Битрикс24 Вайбкод](../../../ai-tools/vibecode.md), чтобы создать приложение для Битрикс24 по описанию задачи без знания языков программирования. Агент напишет код и разместит приложение на сервере без ручной настройки хостинга
- используйте [MCP-сервер](../../../ai-tools/mcp.md), чтобы разрабатывать интеграцию через REST API в своем проекте. Агент будет обращаться к официальной REST-документации

{% endnote %}

Методы раздела помогают управлять личными настройками работы с задачей: включать наблюдение, добавлять задачу в избранное, закреплять ее в списке и включать режим «Без звука».

Эти методы не изменяют содержание задачи, а помогают следить за обновлениями, быстрее находить нужные задачи в списке и отключать лишние уведомления.

> Быстрый переход: [все методы](#all-methods)
> 
> Пользовательская документация: [Как искать избранные задачи](https://helpdesk.bitrix24.ru/open/27648776/#4)

## Как начать работу

1. Получите идентификатор задачи методом [tasks.task.list](../tasks-task-list.md) или [tasks.task.get](../tasks-task-get.md)
2. Включите наблюдение методом [tasks.task.startwatch](./tasks-task-start-watch.md), если хотите получать обновления по задаче
3. Отключите наблюдение методом [tasks.task.stopwatch](./tasks-task-stop-watch.md), если следить за задачей больше не нужно
4. Добавьте задачу в избранное методом [tasks.task.favorite.add](./tasks-task-favorite-add.md), чтобы быстрее находить ее в списке
5. Удалите задачу из избранного методом [tasks.task.favorite.remove](./tasks-task-favorite-remove.md), если она больше не нужна в быстром доступе
6. Закрепите задачу методом [tasks.task.pin](./tasks-task-pin.md), если хотите зафиксировать ее в списке
7. Открепите задачу методом [tasks.task.unpin](./tasks-task-unpin.md), если закрепление больше не требуется
8. Включите режим «Без звука» методом [tasks.task.mute](./tasks-task-mute.md), если по задаче не нужны уведомления
9. Выключите режим «Без звука» методом [tasks.task.unmute](./tasks-task-unmute.md), чтобы снова получать уведомления

## Связь с другими объектами

**Задача.** Все методы раздела работают с существующей задачей по ее идентификатору. Получить идентификатор и данные задачи можно методами [tasks.task.get](../tasks-task-get.md) и [tasks.task.list](../tasks-task-list.md).

**Пользователь.** Методы выполняют действия текущего пользователя по задаче. Возможность выполнить действие зависит от доступа к задаче. Проверить доступ можно методом [tasks.task.getaccess](../tasks-task-get-access.md).

**Список задач.** Закрепление и избранное помогают быстрее находить задачу в общем списке, а режим «Без звука» и наблюдение управляют способом работы с уведомлениями по задаче.

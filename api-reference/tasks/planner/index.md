# Задачи в «Плане на день»: обзор методов

{% note tip "" %}

Выберите инструмент для разработки с AI-агентом:

«План на день» помогает пользователю собрать задачи, дела и встречи, которые нужно выполнить в течение рабочего дня.

Метод раздела получает список идентификаторов задач из «Плана на день» пользователя, от имени которого выполняется запрос. По этим идентификаторам можно получить подробные данные задач и построить рабочий список на день в приложении.

> Быстрый переход: [все методы](#all-methods)
>
> Пользовательская документация: [План на день](https://helpdesk.bitrix24.ru/open/17856080/)

## Как начать работу

1. Получите список идентификаторов задач в плане пользователя, от имени которого выполняется запрос, методом [task.planner.getList](./task-planner-get-list.md)
2. Получите подробные данные нужной задачи методом [tasks.task.get](../tasks-task-get.md)
3. Если нужно отобрать задачи по условиям, используйте метод [tasks.task.list](../tasks-task-list.md)

## Связь с другими объектами

**Задача.** Метод [task.planner.getList](./task-planner-get-list.md) возвращает идентификаторы задач. По идентификатору можно получить данные задачи методом [tasks.task.get](../tasks-task-get.md) или найти задачи по фильтру методом [tasks.task.list](../tasks-task-list.md).

**Пользователь.** Метод работает с планом на день пользователя, от имени которого выполняется запрос. Доступ к данным задач зависит от прав этого пользователя.

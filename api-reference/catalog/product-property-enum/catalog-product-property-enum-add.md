## Описание

```http
catalog.productPropertyEnum.add(fields)
```

Метод добавляет значение списочных свойств.

## Параметры

#|
|| **Параметр** | **Описание** ||
|| **fields**
[`object`](../../data-types.md)| Поля, соответствующие доступному списку полей [`fields`](catalog-product-property-enum-get-fields.md). ||
|#

## Описание

```http
catalog.document.cancelList(documentIds)
```

Метод для групповой отмены проведения документов складского учёта.

## Параметры

#|
|| **Параметр** | **Описание** ||
|| **documentIds**
[`array`](../../data-types.md)| Массив идентификаторов документов, проведение которых требуется отменить. ||
|#

## Описание

```js
catalog.documentcontractor.getFields()
```

Метод возвращает массив полей с привязками.

## Параметры

Без параметров.

## Примеры

{% list tabs %}

- JS

    ```js
    BX.callMethod(
        'catalog.documentcontractor.getFields',
        {},
        function(result) {
            if (result.error())
                console.error(result.error().ex);
            else
                console.log(result.data());
        }
    );
    ```

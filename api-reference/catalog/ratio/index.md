Коэффициент единицы измерения показывает, в каком соотношении учитывается единица товара. Это позволяет корректно пересчитывать количество при обмене данными.

В объекте коэффициента используются:

- `id` — идентификатор коэффициента единицы измерения
- `productId` — идентификатор товара, к которому относится коэффициент
- `ratio` — числовое значение коэффициента единицы измерения
- `isDefault` — признак коэффициента по умолчанию

> Быстрый переход: [все методы](#all-methods)

## Как работать с коэффициентами единиц измерения

1. Получите структуру и типы полей через [catalog.ratio.getFields](./catalog-ratio-get-fields.md).
2. Получите список коэффициентов через [catalog.ratio.list](./catalog-ratio-list.md). Если нужен основной коэффициент товара, выбирайте запись с `isDefault = Y`.
3. Получите данные конкретного коэффициента по его `id` через [catalog.ratio.get](./catalog-ratio-get.md).

## Связь с другими объектами

**Товар.** Коэффициент связан с товаром через поле `productId`. Идентификаторы товаров можно получить методами [catalog.product.list](../product/catalog-product-list.md) и [catalog.product.get](../product/catalog-product-get.md).

# Системные диалоги: обзор методов

{% note tip "" %}

Если вы разрабатываете интеграции для Битрикс24 с помощью AI-инструментов (Codex, Claude Code, Cursor), подключите [MCP-сервер](../../mcp.md), чтобы ассистент использовал официальную REST-документацию.

{% endnote %}

Системные диалоги открываются в интерфейсе Битрикс24 и работают вне фрейма приложения. Их используют, когда нужно выбрать пользователя, права доступа или элементы CRM через стандартные окна платформы.

> Быстрый переход: [все методы](#all-methods)

## Как выбрать диалог

1. Для выбора сотрудников используйте [BX24.selectUser](./bx24-select-user.md) для одного пользователя и [BX24.selectUsers](./bx24-select-users.md) для нескольких.
2. Для выбора прав доступа используйте [BX24.selectAccess](./bx24-select-access.md).
3. Для выбора элементов CRM используйте [BX24.selectCRM](./bx24-select-crm.md).

## Как выглядит рабочий сценарий

1. Определите, какие данные нужны на следующем шаге: ID пользователя, коды доступа или элементы CRM.
2. Передайте обработчик `callback`, чтобы получить выбранные значения после закрытия диалога.
3. Нормализуйте данные из `callback` в нужный формат: один `id`, массив `id` или коды доступа.
4. Используйте полученные значения в следующем действии приложения: REST-вызов, фильтр, настройка прав или заполнение формы.
5. Сохраните результат выбора в состоянии интерфейса, чтобы пользователь мог продолжить сценарий без повторного выбора.

## Связь с другими объектами

**Пользователь.** Методы [BX24.selectUser](./bx24-select-user.md) и [BX24.selectUsers](./bx24-select-users.md) возвращают выбранных сотрудников в обработчик `callback` и помогают передавать их идентификаторы в последующие REST-вызовы.

**Права доступа.** Метод [BX24.selectAccess](./bx24-select-access.md) возвращает коды прав доступа в формате `U1`, `SG4`, `AU` и помогает настраивать правила видимости в объектах, где нужны access-коды.

**CRM.** Метод [BX24.selectCRM](./bx24-select-crm.md) возвращает выбранные элементы CRM по типам (`lead`, `contact`, `company`, `deal`, `quote`) и помогает использовать их в сценариях интеграции.

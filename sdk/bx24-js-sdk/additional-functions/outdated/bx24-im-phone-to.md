# Позвонить на телефонный номер BX24.im.phoneTo

{% note warning "DEPRECATED" %}

Развитие метода остановлено. Используйте [Messenger.startPhoneCall](../messenger-start-phone-call.md).

{% endnote %}

Метод `BX24.im.phoneTo` отправляет команду на звонок по телефонному номеру.

```js
void BX24.im.phoneTo(String phone)
```

## Параметры

{% include [Сноска об обязательных параметрах](../../../../_includes/required.md) %}

#|
|| **Название**
`тип` | **Описание** ||
|| **phone***
`string` | Телефонный номер для звонка. Рекомендуется международный формат (например, `+74951234567`). Также можно передавать номер в локальном формате, например `84012112233` или `8 (495) 711-22-33`. В SDK значение передается как строка без дополнительной валидации ||
|#

## Пример кода

{% include [Сноска о примерах](../../../../_includes/examples.md) %}

```js
BX24.init(function () {
    BX24.im.phoneTo('+74951234567');
});
```

## Обработка ответа

Метод не возвращает данные (`void`).

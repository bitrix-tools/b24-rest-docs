# Изменить размер фрейма BX24.resizeWindow

{% note tip "" %}

Выберите инструмент для разработки с AI-агентом:

- используйте [Битрикс24 Вайбкод](../../../ai-tools/vibecode.md), чтобы создать приложение для Битрикс24 по описанию задачи без знания языков программирования. Агент напишет код и разместит приложение на сервере без ручной настройки хостинга
- используйте [MCP-сервер](../../../ai-tools/mcp.md), чтобы разрабатывать интеграцию через REST API в своем проекте. Агент будет обращаться к официальной REST-документации

{% endnote %}

Метод `BX24.resizeWindow` отправляет команду на изменение размера фрейма приложения по переданным ширине и высоте.

```js
void BX24.resizeWindow(Integer width, Integer height[, Function callback])
```

## Параметры

{% include [Сноска об обязательных параметрах](../../../_includes/required.md) %}

#|
|| **Название**
`тип` | **Описание** ||
|| **width***
`integer` | Ширина фрейма в пикселях. Внутри метода значение приводится через `parseInt` и используется только если больше `0` ||
|| **height***
`integer` | Высота фрейма в пикселях. Внутри метода значение приводится через `parseInt` и используется только если больше `0` ||
|| **callback**
`function` | Функция обратного вызова, выполняется после отправки команды изменения размера ||
|#

## Пример кода

{% include [Сноска о примерах](../../../_includes/examples.md) %}

```js
BX24.init(function () {
    BX24.resizeWindow(980, 700, function () {
        console.log('Команда на изменение размера фрейма отправлена');
    });
});
```

## Обработка ответа

Метод не возвращает данные (`void`).

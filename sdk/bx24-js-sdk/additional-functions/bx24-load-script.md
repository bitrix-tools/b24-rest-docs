# Загрузить javascript-файл BX24.loadScript

{% note tip "" %}

Выберите инструмент для разработки с AI-агентом:

- используйте [Битрикс24 Вайбкод](../../../ai-tools/vibecode.md), чтобы создать приложение для Битрикс24 по описанию задачи без знания языков программирования. Агент напишет код и разместит приложение на сервере без ручной настройки хостинга
- используйте [MCP-сервер](../../../ai-tools/mcp.md), чтобы разрабатывать интеграцию через REST API в своем проекте. Агент будет обращаться к официальной REST-документации

{% endnote %}

Метод `BX24.loadScript` загружает и выполняет один или несколько javascript-файлов. Если передан массив, файлы загружаются последовательно. Если DOM еще не готов, то есть страница еще не разобрана браузером и элементы еще недоступны для скрипта, выполнение откладывается до [BX24.ready](./bx24-ready.md).

```js
void BX24.loadScript(Array|String script[, Function callback])
```

## Параметры

{% include [Сноска об обязательных параметрах](../../../_includes/required.md) %}

#|
|| **Название**
`тип` | **Описание** ||
|| **script***
`array|string` | Путь к javascript-файлу или массив путей. При передаче массива файлы загружаются и выполняются по очереди ||
|| **callback**
`function` | Функция обратного вызова, которая выполняется после завершения загрузки всех файлов ||
|#

## Пример кода

{% include [Сноска о примерах](../../../_includes/examples.md) %}

```js
BX24.loadScript(
    [
        '/local/scripts/first.js',
        '/local/scripts/second.js'
    ],
    function () {
        console.log('Все скрипты загружены');
    }
);
```

## Обработка ответа

Метод не возвращает данные (`void`).

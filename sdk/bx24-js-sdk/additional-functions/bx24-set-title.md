# Установить заголовок страницы BX24.setTitle

{% note tip "" %}

Выберите инструмент для разработки с AI-агентом:

- используйте [Битрикс24 Вайбкод](../../../ai-tools/vibecode.md), чтобы создать приложение для Битрикс24 по описанию задачи без знания языков программирования. Агент напишет код и разместит приложение на сервере без ручной настройки хостинга
- используйте [MCP-сервер](../../../ai-tools/mcp.md), чтобы разрабатывать интеграцию через REST API в своем проекте. Агент будет обращаться к официальной REST-документации

{% endnote %}

Метод `BX24.setTitle` отправляет команду на изменение заголовка страницы приложения.

```js
void BX24.setTitle(String title[, Function callback])
```

Mетод `BX24.setTitle` меняет заголовок внутри контейнера приложения: страницы или слайдера. Заголовком верхней вкладки браузера управляет страница портала, поэтому команда может выполниться успешно, но заголовок вкладки браузера не изменится.

## Параметры

{% include [Сноска об обязательных параметрах](../../../_includes/required.md) %}

#|
|| **Название**
`тип` | **Описание** ||
|| **title***
`string` | Новый заголовок страницы. Внутри метода значение приводится к строке через `toString()` ||
|| **callback**
`function` | Функция обратного вызова, выполняется после отправки команды на изменение заголовка ||
|#

## Пример кода

{% include [Сноска о примерах](../../../_includes/examples.md) %}

```js
BX24.init(function () {
    BX24.setTitle('Новый заголовок', function () {
        console.log('Команда на изменение заголовка отправлена');
    });
});
```

## Обработка ответа

Метод не возвращает данные (`void`).

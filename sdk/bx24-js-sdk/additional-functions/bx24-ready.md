# Установить обработчик готовности DOM-структуры BX24.ready

{% note tip "" %}

Если вы разрабатываете интеграции для Битрикс24 с помощью AI-инструментов (Codex, Claude Code, Cursor), подключите [MCP-сервер](../../mcp.md), чтобы ассистент использовал официальную REST-документацию.

{% endnote %}

Метод `BX24.ready` добавляет функцию-обработчик, которая выполняется после готовности DOM-структуры документа. То есть обработчик запустится, когда страница уже разобрана браузером и ее элементы доступны для скрипта. Метод действует аналогично `jQuery.ready` или `BX.ready`. Если в параметре передать не функцию, вызов будет проигнорирован.

```js
void BX24.ready(Function handler)
```

## Параметры

{% include [Сноска об обязательных параметрах](../../../_includes/required.md) %}

#|
|| **Название**
`тип` | **Описание** ||
|| **handler***
`function` | Функция-обработчик, которая будет вызвана после готовности DOM-структуры документа ||
|#

## Пример кода

{% include [Сноска о примерах](../../../_includes/examples.md) %}

```js
BX24.ready(function () {
    console.log('DOM готов');
});
```

## Обработка ответа

Метод не возвращает данные (`void`).

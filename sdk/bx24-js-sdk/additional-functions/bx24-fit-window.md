# Подстроить размер фрейма под содержимое BX24.fitWindow

{% note tip "" %}

Если вы разрабатываете интеграции для Битрикс24 с помощью AI-инструментов (Codex, Claude Code, Cursor), подключите [MCP-сервер](../../mcp.md), чтобы ассистент использовал официальную REST-документацию.

{% endnote %}

Метод `BX24.fitWindow` отправляет команду на изменение высоты фрейма приложения по высоте контента страницы. В реализации используется `BX24.getScrollSize().scrollHeight`.

```js
void BX24.fitWindow([Function callback])
```

## Параметры

#|
|| **Название**
`тип` | **Описание** ||
|| **callback**
`function` | Функция обратного вызова, выполняется после отправки команды изменения размера фрейма ||
|#

## Пример кода

{% include [Сноска о примерах](../../../_includes/examples.md) %}

```js
BX24.init(function () {
    BX24.fitWindow(function () {
        console.log('Команда на изменение размера фрейма отправлена');
    });
});
```

## Обработка ответа

Метод не возвращает данные (`void`).

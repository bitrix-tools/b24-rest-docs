# Перезагрузить страницу BX24.reloadWindow

{% note tip "" %}

Если вы разрабатываете интеграции для Битрикс24 с помощью AI-инструментов (Codex, Claude Code, Cursor), подключите [MCP-сервер](../../mcp.md), чтобы ассистент использовал официальную REST-документацию.

{% endnote %}

Метод `BX24.reloadWindow` отправляет команду на перезагрузку страницы с приложением. Начиная с версии модуля `rest` `25.800.0` метод можно использовать в [местах встраивания](../../../api-reference/widgets/index.md) приложения.

```js
void BX24.reloadWindow([Function callback])
```

## Параметры

#|
|| **Название**
`тип` | **Описание** ||
|| **callback**
`function` | Функция обратного вызова, выполняется после отправки команды на перезагрузку страницы ||
|#

## Пример кода

{% include [Сноска о примерах](../../../_includes/examples.md) %}

```js
BX24.init(function () {
    BX24.reloadWindow(function () {
        console.log('Команда на перезагрузку страницы отправлена');
    });
});
```

## Обработка ответа

Метод не возвращает данные (`void`).

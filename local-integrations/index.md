# Инструменты для локальных интеграций

Локальные интеграции — это программные решения, которые создаются и настраиваются на конкретном Битрикс24. В отличие от облачных приложений из Маркетплейса, локальные инструменты предназначены для внутренних задач компании и могут быть адаптированы под специфические бизнес-процессы.

## Типовые задачи для локальных интеграций

Используйте локальные инструменты для решения следующих задач:

- импорт данных в CRM или другие инструменты Битрикс24
- интеграция с внутренними системами
- простая автоматизация, связанная с массовой обработкой данных клиентов
- встраивание кастомных элементов в интерфейс Битрикс24

## Входящие и исходящие вебхуки

[Входящие и исходящие вебхуки](./local-webhooks.md) подходят для быстрых интеграций и простых сценариев, когда не требуется сложная логика авторизации. Входящие вебхуки предоставляют бессрочный ключ для вызова методов REST API, исходящие — отправляют данные во внешние системы при наступлении событий в Битрикс24.

{% note warning "" %}

Для работы исходящего вебхука в коробочной версии Битрикс24 нужна активная лицензия, на демо-режимах исходящий вебхук недоступен

{% endnote %}

## Локальные приложения

[Локальные приложения](./local-apps.md) позволяют:

- создавать пользовательский интерфейс
- выполнять серверную обработку
- подписываться на события
- настраивать права доступа
  
Существует три вида локальных приложений:

- [Статичное приложение](./static-local-app.md) — работает в браузере на HTML/JS, не требует своего сервера, отображается как отдельная страница в интерфейсе Битрикс24.
- [Серверное приложение с интерфейсом](./serverside-local-app-with-ui.md) — код выполняется на вашем сервере, отображается как страница или встраиваемый виджет внутри Битрикс24.
- [Серверное приложение без интерфейса](./serverside-local-app-with-no-ui.md) — работает в фоне на вашем сервере, не отображается в интерфейсе, подходит для автоматических задач и синхронизации.

## Раздел Разработчикам

В разделе [Разработчикам](./developers-area.md) собраны инструменты для упрощения работы с локальными интеграциями. Попасть в раздел можно через левое меню Битрикс24 по пути *Приложения > Разработчикам*.

### Готовые сценарии

Шаблоны для типовых задач с примерами кода и предустановленными параметрами на базе вебхуков и локальных приложений.

### Интеграции

Список всех созданных вебхуков и приложений с информацией о правах доступа, событиях и виджетах.

### Статистика

График общего количества запросов и детализация по каждой интеграции. Помогает контролировать нагрузку и отлаживать работу.
